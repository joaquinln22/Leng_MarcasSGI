Propiedades de Espaciado y Tamaño

  • margin: Establece el espacio exterior de un elemento.
    Ejemplo: margin: 5px 20px 5px 0;
  • padding: Define el espacio interior entre el contenido y el borde.
    Ejemplo: padding: 3px;
  • width: Establece el ancho de un elemento.
    Ejemplo: width: 1100px;
  • height: Define la altura de un elemento.
    Ejemplo: height: 30px;

Propiedades de Texto y Fuente

  • Insertar fuente en html: <link href="url_google_font" rel="stylesheet">
  • font-family: Especifica la fuente de texto en un elemento.
    Ejemplo: font-family: 'Open Sans', sans-serif;
  • font-size: Define el tamaño de la fuente.
    Ejemplo: font-size: 0.95em;
  • font-weight: Controla el grosor de la fuente.
    Ejemplo: font-weight: bold;
  • color: Establece el color del texto.
    Ejemplo: color: white;
  • text-align: Alinea horizontalmente el texto dentro de un elemento.
    Ejemplo: text-align: center;
  • text-decoration: Controla decoraciones de texto (por ejemplo, subrayado).
    Ejemplo: text-decoration: none;
  • @font-face: Insertar una fuente específica.
    Ejemplo: @font-face{ font-family: "MiFuente"; src: url(""); };

Propiedades de Fondo y Bordes

  • background-color: Define el color de fondo del elemento.
    Ejemplo: background-color: darkgrey;
  • linear-gradient: Degradado lineal del color de fondo del elemento.
    Ejemplo: background: linear-gradient(top, #ffffff, #000000);
  • radial-gradient: Degradado circular del color de fondo del elemento.
    Ejemplo: background: radial-gradient(center, circle, #ffffff, #000000);
  • border: Establece un borde alrededor de un elemento.
    Ejemplo: border: 1px solid black;
  • border-radius: Redondea las esquinas del borde.
    Ejemplo: border-radius: 5px;

Modelo de Caja (Box Model)

  • box-shadow: Aplica una sombra dentro o fuera de un elemento.
    Ejemplo: box-shadow: 0px 5px 5px 0px rgba(0, 0, 0, 0.5);
  • clear: Controla el comportamiento de los elementos flotantes, evitando que otros elementos se alineen al lado del elemento actual.
    Ejemplo: clear: both;

Visualización y Posicionamiento

  • float: Controla la alineación de un elemento dentro de su contenedor (a la izquierda o derecha).
    Ejemplo: float: left;
  • display: Define cómo debe mostrarse el elemento (por ejemplo, en línea o como bloque).
    Ejemplo: display: block;
    Ejemplo: display: flex;
  • flex-direction: que cambia la orientación del eje principal. Puede tomar los valores:
    • row
    • row-reverse
    • column
    • column-reverse
  • flex-wrap: que evita o permite el desbordamiento (multilínea). Puede tomar los valores:
    • nowrap
    • wrap
    • wrap-reverse
  • justify-content: que se utiliza para alinear los ítems del eje principal (por defecto, el horizontal). Puede tomar los valores:
    • flex-start
    • flex-end
    • center
    • space-between
    • space-around
    • space-evenly
  • align-items: usada para alinear los ítems del eje secundario (por defecto, el vertical). Puede tomar los valores:
    • flex-start
    • flex-end
    • center
    • space-between
    • space-around
    • stretch
  • transform: Permite la transformación de un objeto.
  • transform:scale: Realiza una transformación de escala.
    Ejemplo: .imagen:hover{ -webkit-transform: scale(3); }
  • transform:rotate: Realiza una transformación de giro.
    Ejemplo: .imagen:hover{ -webkit-transform: rotate(10deg); }
  • transform:translate: Realiza una transformación de cambio de posición.
    Ejemplo: .imagen:hover{ -webkit-transform: translateX(100px) translateY(100px); }
  • transform:skew: Realiza una transformación de distorsión.
    Ejemplo: .imagen:hover{ -webkit-transform: skew(20deg, 10 deg); }
  • transition: Modifica parámetros como el orden y velocidad de las transiciones.
    Ejemplo: .imagen:hover{ -webkit-transition: -webkit-transform 1s ease-in-out 0.5s;}

Pseudo-clases

  • :hover: Aplica estilos cuando el cursor pasa sobre el elemento.
    Ejemplo: a:hover { background-color: darkgray; }

Propiedades de Listas y Enlaces

  • list-style: Define el tipo de marcador de lista (none, circle, number, square, lower-alpha).
    Ejemplo: list-style: none;
  • text-decoration: Controla decoraciones en enlaces (overline, underline, line-through).
    Ejemplo: text-decoration: none;
